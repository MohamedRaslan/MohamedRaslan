# UR10 Pick And Place

## Summary

UR10 Pick And Place is an early robotics project using MATLAB and V-REP to simulate a simple robotic arm pick-and-place flow.

## Problem

Robotics projects combine logic, movement, constraints, and simulation. Even a small project can show how software thinking applies outside classic web or backend systems.

## What It Shows

- MATLAB scripting.
- Simulation-driven experimentation.
- Curiosity across robotics, automation, and engineering systems.

## Quality Angle

This project adds personality to the profile because it shows range: automation is not only browser testing or APIs, it can also touch physical systems and robotics thinking.
