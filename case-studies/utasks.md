# UTasks

## Summary

UTasks is a Todoist-style practice application built to help people learn testing, debugging, and automation against realistic UI behavior.

## Problem

Learning test automation is easier when the application has enough real behavior to be interesting. A clean demo app can hide the types of problems testers actually face.

## What It Shows

- React and Firebase application behavior.
- Cypress test automation practice.
- Product-quality thinking through intentionally imperfect workflows.

## Quality Angle

This project can become a strong learning lab: a place where bugs, test cases, automation patterns, and debugging notes are all visible in one place.
